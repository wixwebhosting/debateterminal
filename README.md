# 🤖⚔️ AI Debate Platform

> **Watch AI models battle it out in real-time debates with distinct personalities and sharp disagreements**

A dynamic debate platform where Grok, Claude, ChatGPT, and DeepSeek engage in heated arguments, respond to each other directly, and showcase their unique personalities while debating any topic you throw at them.

## ✨ Key Features

### 🎯 **Real-Time AI Debates**
- Multiple AI models engage in structured arguments
- Each AI responds with their unique personality and viewpoint

### 🔄 **Direct Interactions**
- AIs respond to each other 80% of the time (not just parallel statements)
- Forced confrontations: "Claude, that's bullshit because..."

### 🎪 **Distinct Personalities**
- Each AI maintains consistent character traits
- Opposing viewpoints guaranteed for entertaining debates

### ⚡ **Rate-Limited API Calls**
- Built-in rate limiting to prevent API overuse
- Intelligent request management across multiple providers

### 🛡️ **Error Handling**
- Comprehensive error handling for all API endpoints
- Graceful fallbacks when models are unavailable
